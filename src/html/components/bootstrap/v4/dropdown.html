{# Modifications Trigger dropdown menus above elements by adding .dropup to the parent element (e.g. div class="btn-group dropup"). The same with .dropright, .dropleft Add .dropdown-menu-right to a .dropdown-menu to right align the dropdown menu. If you want to use responsive alignment, disable dynamic positioning by adding the data-display="static" attribute and use the responsive variation classes. To align right the dropdown menu with the given breakpoint or larger, add .dropdown-menu{-sm|-md|-lg|-xl}-right. #}
